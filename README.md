# инсталятион

нада:
- комп
- OpenOS (установленная, а не с дискеты загруженная)
- мозг, руки (не из жопы), ноги (из жопы), жопа (не с руками, а с ногами)

пишете cd, mkdir куда вам надо и вставляете:

```bash
wget https://pastebin.com/raw/r7xzRRnx && rename r7xzRRnx main
```

всё! появиться файл main! зопустите ёго:

```bash
main
```

или:

```bash
./main
```

как хотите так и запускайте крч, там дохуя вариантов

на этом всё ставьте лайки и подписывайтесь на ганал
